// 关于

什么是 PaiCLI？

🎯

对标 Claude Code

以 Claude Code 为产品标杆，打造 Java 生态下最强大的终端 Agent。从第 1 期 ReAct 单代理循环，一路演进到第 16 期 TUI 产品化。

☕

纯 Java 实现

基于 Java 17 + Maven 构建，零外部运行时依赖。OkHttp 做 HTTP 通信，Jackson 序列化，JLine3 做终端交互，SQLite 持久化。

🧠

多模型支持

内置 GLM-5.1（200K）、DeepSeek V4（1M）、StepFun（256K）三个 LLM 客户端，/model 命令运行时热切换。

🛡️

安全可审计

HITL 人工审批 + 路径围栏 + 命令黑名单 + 操作审计。不是沙箱，但提供生产可用的安全模型。

// 能力

核心特性

16.1 期已完成，21 期路线持续扩展 Agent 产品能力矩阵

01

ReAct 循环

思考 → 行动 → 观察，支持流式推理展示、工具调用、对话历史管理、长上下文自动压缩。

流式输出工具调用上下文压缩
02

Plan & Execute

复杂任务先规划再执行，DAG 依赖拓扑排序，并行批次推进，支持用户审阅与补充。

DAG并行执行人工审阅
03

Multi-Agent 协作

规划者 + 执行者 + 检查者，主从架构编排，审查不过自动重试，Worker 并发执行。

编排器角色分工质量审查
04

Memory 系统

短期记忆 + 长期记忆 + 摘要压缩 + 事实提取。跨会话复用稳定知识，智能检索相关上下文。

长期记忆摘要压缩事实提取
05

RAG 检索

代码向量化 + SQLite 向量存储 + 语义检索 + AST 关系图谱。自然语言搜代码，理解代码结构。

Embedding语义搜索代码图谱
06

HITL 审批

危险操作（写文件 / 执行命令 / 创建项目）人工审批。单字符交互：批准 / 放行 / 拒绝 / 跳过 / 改参。

安全审批路径围栏操作审计
07

MCP 协议

支持 stdio 子进程 + Streamable HTTP 双传输。自动注册 60+ MCP 工具，resources 双轨与 @-mention。

stdioStreamable HTTPresources
08

Chrome DevTools

集成 chrome-devtools-mcp，支持导航 / 快照 / 点击 / 填表 / 脚本执行。CDP 会话复用，登录态访问。

浏览器自动化登录态复用SPA 渲染
09

联网能力

web_search（智谱 / SerpAPI / SearXNG）+ web_fetch（readability 提取 → Markdown）。安全策略内置。

联网搜索网页抓取安全策略
10

TUI 产品化

三种 Renderer：inline 流式（默认）/ lanterna 全屏 / plain 兜底。行内折叠、diff、状态栏。

inline TUIlanterna行内 diff
11

Skill 系统

从硬编码提示词中抽出可复用单元。SKILL.md 决策手册 + references 资料库 + 内置 web-access skill。

可复用三层加载站点经验
12

长上下文工程

short / balanced / long 三种模式，自适应 RAG topK、MCP 索引注入。1M token window 支持。

1M context自适应prompt cache
// 演进

21 期路线进度

从 ReAct 到完整 Agent 产品，每一期都是质的飞跃

01

ReAct Agent CLI

单轮对话驱动的 ReAct 循环，支持文件操作、Shell 命令、项目创建。基础工具集建立。

02

Plan-and-Execute + DAG

复杂任务规划能力。先拆解、再按依赖顺序执行，支持计划确认与补充。

03

Memory + 上下文工程

短期 / 长期记忆、摘要压缩、事实提取、Token 预算管理。跨会话知识复用。

04

RAG 检索 + 代码库理解

代码向量化、SQLite 向量存储、语义检索、AST 关系图谱。自然语言搜代码。

05

Multi-Agent 协作

规划者 + 执行者 + 检查者，主从编排。审查未通过自动重试，Worker 并发。

06

HITL 人工审批

危险操作审批流。路径围栏、命令黑名单、操作审计。三级危险等级。

07

异步执行 + 并行工具

同一轮多工具并行执行。Plan / Team 按依赖批次并发。统一超时与取消。

08

多模型适配

GLM-5.1 / DeepSeek V4 / StepFun 三模型。/model 运行时热切换。

09

联网能力

web_search（三 provider）+ web_fetch（readability 提取）。内置安全策略。

10

MCP 协议核心

stdio + Streamable HTTP 双传输。自动注册工具，schema 清洗。

11

MCP 高级能力

resources 双轨、prompts 查看、@-mention、被动通知、运行中取消。

12

长上下文工程

三种上下文模式、动态预算、prompt cache 可见化。DeepSeek V4 1M window。

13

Chrome DevTools MCP

集成 chrome-devtools-mcp。浏览器导航、快照、点击、填表。SPA / 防爬墙覆盖。

14

CDP 会话复用

登录态访问、敏感页面策略、/browser 命令组。复用日常 Chrome。

15

Skill 系统

SKILL.md 决策手册 + references 资料库。内置 web-access skill，站点经验可积累。

16

TUI 产品化

Renderer 接口 + 三种实现（inline / lanterna / plain）。行内折叠、diff、状态栏。

17

LSP 诊断注入 进行中

Agent 改完代码后，自动收集编译诊断并在下一轮推理前注入。

18

Git Side-History 快照与回滚 计划

Turn 粒度自动快照，支持不污染用户 Git 历史的一键恢复。

19

Prompt 分层架构 计划

把硬编码 system prompt 拆成可维护、可覆盖的 Markdown 分层。

20

异步后台任务 + Runtime API 计划

持久化任务队列与 HTTP/SSE Runtime API，面向无头和集成场景。

21

多模态 LLM 输入 计划

在安全网和架构就绪后，再接通截图、图片和图表的 vision 输入。

快速安装

Java 17 + Maven，三分钟跑起来

1

配置 API Key

terminal
cp .env.example .env
# 编辑 .env，填入 API Key
GLM_API_KEY=your_api_key_here
DEEPSEEK_API_KEY=your_deepseek_key_here
STEP_API_KEY=your_step_key_here
2

编译运行

terminal
# 编译打包
mvn clean package

# 启动 PaiCLI
java -jar target/paicli-1.0-SNAPSHOT.jar

# 或直接运行（无需先 package）
mvn clean compile exec:java \
  -Dexec.mainClass="com.paicli.cli.Main"
3

开始使用
